Functions

getGameId ()
Get the players game identifier.

Returns:

    string The game id the player belongs to.

Usage:

    local game_id = client:getGameId()
getHost ()
Get the client host address.

Returns:

    string Host address.

Usage:

    local host_addr = client:getHost()
getId ()
Get the Client /Player identifier.

Returns:

    string player_id.
getPlayerData ()
Get the players custom data table. Custom data can be included at connection time or during runtime by using setPlayerData .

Returns:

    table The custom Client /Player data.

Or

    nil
getPlayerHandle ()
Gets the players connection handle. This can represent a name, or any other identifier.

Returns:

    string The Client /Players name.

Usage:

    local player_handle = client:getPlayerHandle()
getPlayerNum ()
Get the players game position number.

Returns:

    int Client /Player position.

Usage:

    local player_num = client:getPlayerNum()
getPort ()
Get the client port.

Returns:

    int Port number.

Usage:

    local port_num = client:getPort()
isInGame ()
Checks to see if the Client /Player is in a Game

Returns:

    boolean

Usage:

    local in_game = client:isInGame()
send (data_tbl)
Send data to the Client.

Parameters:

  • data_tbl table A table of data to send.

Usage:

    client:send( { points = 12, update = 1 } )
setPlayerData (data)
Set the players custom data table.

You should be sure to get the current data table using getPlayerData and make your changes on that table before using this method.

Parameters:

  • data table The player data table to save/update.

Usage:

     --== Get the current player data
 local pd = client:getPlayerData()
 --== Do work
 pb.fav_color = "Red"
 --== Store it
 client:setPlayerData( pb )
 --== Clear player data
 client:setPlayerData( {} )
setTimeout (seconds)
Set the Client timeout.

Parameters:

  • seconds int The seconds before timeout occurs.

Usage:

    client:setTimeout( 300 )
